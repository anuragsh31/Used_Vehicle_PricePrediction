## Used_Vehicle_PricePrediction

In this project, Predict the price of vehicles. Car dataset contains information about used cars.

The columns in the given dataset are as follows:

1. name
2. year
3. selling_price
4. km_driven
5. fuel
6. seller_type
7. transmission
8. Owner
